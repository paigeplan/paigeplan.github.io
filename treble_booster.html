Rangemaster Treble Booster

I got the schematic for this pedal fuzzcentral's site (http://fuzzcentral.ssguitar.com/rangemaster.php) Instead of making it on perf board, I created a PCB layout in Eagle so that I could mill it out with my school's PCB miller.

Here's what it looks like in Eagle. You first create a schematic to specify how each component should be electrically connected, then you switch to the another mode to layout how the board should be physically laid out.

I then uploaded that file to this milling machine (this one is an Othermill), which milled and drilled a blank copper board into the PCB I designed.

And here is that same PCB populated with components. I later added a light.

If you'd like to mill your own, you can use my .brd file I uploaded here: https://github.com/paigeplan/rangemaster.
